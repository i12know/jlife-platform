# Research Archive

Raw research inputs, kept verbatim for provenance and future re-derivation.

**Status rules — read before ingesting anything in this directory:**

1. **Nothing here is authoritative.** These are inputs (deep-research LLM
   reports, external analyses, meeting notes) preserved as received. Claims are
   unverified unless a curated doc says otherwise. When a file here conflicts
   with `docs/` proper, the curated doc wins.
2. **Decisions never live here.** Each archived file's header names the
   decision(s) it fed and links to the curated doc that records the outcome.
   To understand the platform, read the curated corpus; come here only to
   audit *why* a decision went the way it did, or to re-derive it when
   circumstances change.
3. **LLM sessions:** do not load this directory by default. Read a file here
   only when a task explicitly concerns re-examining the decision it fed.
4. **Naming:** `YYYY-MM-DD-topic.md`, dated by receipt. Files are immutable
   after archiving except for header corrections — new research gets a new
   dated file, never an edit to an old one.
5. **Repo safety (PRD §21) still applies:** no partner-identifying data,
   credentials, or private-system exports may be archived, even as "research."
